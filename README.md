# Youcef Yaich — Portfolio

One-page portfolio built with vanilla HTML, CSS & JavaScript. No frameworks, no build tools.

**Live:** [youcefffj.github.io/My_Resume_CV](https://youcefffj.github.io/My_Resume_CV)

## Features

- Terminal hero with typing animation & matrix rain background
- Experience timeline
- Dynamic project cards fetched from GitHub API
- Inline CV viewer (FR / EN) with language switcher
- Spotify "Now Playing" widget via last.fm
- Fully responsive, dark cyber/futuristic theme

## Stack

HTML / CSS / JavaScript — GitHub Pages

## Configuration

Edit `config.json` to customize:

- **Terminal** — commands and outputs displayed in the hero
- **Projects** — override titles, descriptions, badges for GitHub repos
- **Status** — auto-detects recent repos + fixed items
- **Spotify** — last.fm API key and username

Projects are fetched automatically from GitHub. New public repos appear on the portfolio without any code change.
